싱글톤 패턴이란?
--------------------------------
1. 프로그램에서 인스턴스가 단 한개만 생성되어야 하는 경우 사용하는 디자인 패턴 //날짜같은거, 사원객체는 많지만, 회사라는 객체는 1개임  
2. static 변수, 메서드를 활용하여 구현할 수 있음.

싱글톤 패턴으로 회사 객체 구현하기
----------------------------
1. 생성자는 private로 선언  
private Company() {}  
2. 클래스 내부에 유일한 private 인스턴스 생성
private static Company instance = new Company();
